Cara Upload ke GitHub (tanpa terminal)
⬆ Upload file lewat GitHub.com — step by step
1
Buka repository kamu
Pergi ke github.com/mlnhdyt/damkar-bojonegoro dan pastikan sudah login.
2
Klik "Add file" → "Upload files"
Ada di pojok kanan atas daftar file, di sebelah tombol hijau Code.
📸 Tombol "Add file" ada di baris yang sama dengan nama branch
3
Drag & drop atau pilih file
Seret file sidata-links.html dan semua gambar ke area upload, atau klik "choose your files". Bisa pilih banyak sekaligus.
4
Isi commit message (opsional tapi bagus)
Di bagian bawah ada kolom "Commit changes". Isi singkat, contoh: tambah halaman sidata-links
5
Klik "Commit changes" warna hijau
Tunggu beberapa detik sampai halaman refresh dan file kamu muncul di daftar.
6
Akses halaman baru kamu
Buka browser, ketik: mlnhdyt.github.io/damkar-bojonegoro/sidata-links.html — tunggu ±1–2 menit kalau belum muncul, GitHub Pages butuh waktu deploy.
